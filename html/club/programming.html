Club Overview

The club offers a steady space to learn, build, and discuss technology in a practical way. Sessions often combine guided teaching, short exercises, project work, and peer feedback so members can turn ideas into usable portfolio pieces.

Venue Information

  • Online meetings: Most weekly sessions run online, with links shared through club notices and the study group.
  • Optional in-person meetups: Project check-ins or workshops may take place in library study zones or computer labs.
  • Shared materials: Code samples, slides, and task lists are usually organised after each activity.
  • First-time joiners: Joining the call a few minutes early helps confirm audio, video, and account access.
  • Follow-up: Members can continue discussion and collaboration through shared project spaces.

What We Do

  • Foundations: HTML, CSS, JavaScript, and practical frontend work
  • Framework learning: Vue concepts such as components and routing
  • Programming: Python scripting, automation, and problem solving
  • Projects: small-team builds, review cycles, and demo preparation

Who It Is For

  • Beginners who want a structured entry point into programming
  • Members building stronger project, tooling, or engineering skills
  • Students creating portfolio work or collaborative side projects
  • Anyone curious about AI, web, or data-related topics

Session Flow (Sample)

  • Teaching block: topic overview, demo, and guided explanation
  • Hands-on practice: short exercises or debugging tasks
  • Project time: pair work or team-based build progress
  • Review and Q&A: code discussion, next steps, and resource sharing

Learning Notes

  • Bring your own laptop and tools for in-person workshops when announced
  • Questions are welcome at any level, especially during practical segments
  • Shared examples and task lists are normally posted after the session
  • In-person meetups, if arranged, are announced with location and timing details in advance
